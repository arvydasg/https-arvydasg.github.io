1. Intro

Why not track everything about myself?

Instead of writing a manual journal each evening, why not pull various info from API's of the apps that I am using daily? I am big on tracking things, storing things, reflecting on things after some time.. this app will be perfect for that.

2. Metrics tracked

  • Daily quote
  • Daily weather
  • Rescuetime daily computer usage
  • Various personal questions

On a daily basis, some records about the day would be fetched from various public API's and displayed on the home page. Some stats about my previous day would be fetched and put into the database. I would answer questions about my day during the evenings, this information also get's stored to the database.

3. Inspiration for this app
