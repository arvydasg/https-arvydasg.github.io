django commands

Create django project 

django-admin startproject mysite

Create django app 

python manage.py startapp appname

Create superuser 

python manage.py createsuperuser

python manage.py makemigrations

python manage.py migrate
python manage.py migrate appname

python manage.py runserver

If you have deleted migrations folder or it's content, running makemigrations or migrate will not do anything. First need to run this: https://stackoverflow.com/questions/34548768/no-such-table-exception 

python manage.py migrate --run-syncdb

Django shell.

This opens up a terminal, but it has the Django settings already imported, so it allows you to work directly from the root folder of a Django project. 

python manage.py shell

Fetch opt values. Write this in the template 

<script>
 console.log("Model name: {{ opts.model_name }}");
</script>

Move the admin fields over js example(shipping): 

const manual_ship_type_field = document.querySelector('div.field-manual_ship_type');
const whole_manual_mapping_block = manual_ship_type_field.parentElement.parentElement;
const forecasted_values_block = document.getElementById('vessel_vessel_forecasted_value-group');
whole_manual_mapping_block.insertAdjacentElement('afterend', forecasted_values_block);
