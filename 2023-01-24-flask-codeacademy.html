1. 2023.01.24

"Flask yra populiariausias Python microframework. Jeigu projektas nėra labai didelis, arba tiesiog norime pasidaryti kažkokį GUI per naršyklę, kažką greitai prototipuoti, Flask yra labai geras pasirinkimas. Su flask yra pakankamai paprasta kurti API's. Ši paskaita bus greita įžanga, kitose nagrinėsime išsamiau."

pip install flask

create main.py

export FLASK_APP=main.py

python -m flask run

template_folder = "render" - import this to use template from other templates folder, prie name

if you want to change templates folder name - don't forget to add "defaulttemplates" to app = in main.

def home():

if __name__ == "__main__":
    app.run(debug=True):

1.2. Kaip puslapyje atvaizduoti įvestą kintamąjį:

@app.route("/<name>")

    return f"Labas, {name}"

if __name__ == "__main__":
    app.run()

1.5. Kaip perduoti duomenis iš svetainės į programą:

app.py faile: 

from flask import Flask, request, render_template
app = Flask(__name__)

@app.route("/login", methods=['GET', 'POST'])
def login():
    if request.method == "POST":  # if post request(post request from form)
        vardas = request.form['name_ane']  # take the value from name_ane and put it in "vardas" variable
        return render_template("greetings.html", vardas=vardas)  # render greetings template where vardas variable bus paduotas kaip vardas
    else:
        return render_template("login.html")


if __name__ == "__main__":
    app.run()

templates/login.html faile: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Prisijungimas</title>
</head>
<body>
<form action="#" method="post"> <!-- post metodas, okei -->
    <p>Vardas:</p>
    <p><input type="text" name="name_ane"/></p> <!-- name_ane -->
    <p><input type="submit" value="submit"/></p> <!-- upon submit renderinam greetings.html -->
</form>
</body>
</html>

templates/greetings.html faile: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Sveikiname</title>
</head>
<body>
<h1>{{vardas}}, sveikiname prisijungus!</h1> <!-- vardas=vardas, del to ir galim pasiekti -->
</body>
</html>
