Create and implement base django template

https://stackoverflow.com/questions/14720464/django-project-base-template

Create base.html file in your templates directory: 

<!DOCTYPE html>
<html>
<head>
    <title>Project</title>
</head>
<body>
{% block content %}

{% endblock %}
</body>
</html>

extend it: 

{% extends "base.html" %}

{% block content %}
    <h2>Content for My App</h2>
    <p>Stuff etc etc.</p>
{% endblock content %}
