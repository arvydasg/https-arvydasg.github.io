1 Regular user windows commands

Lots of shortcuts already in my head, but some that are new are: 

win + ctrl + t          # always on top window
alt + space                     # look for programs
win + shift + /                 # ALL SHORTCUTS
win+shift+T             # TEXT EXTRACTOR WTF??!!
win + alt + n           # new quick note in onenote
win + z                         # choose layouts

2 Admin user windows commands

net group AD-GROUP-NAME /domain # see members of the AD group
net user usernamexx /domain     # check your own group membership
echo %computername%             # find computer name
wmic bios get serialnumber      # find computer's serial number

3 Additional tips

Install powertoys app in windows for rebinding various keys, especially caps -> ctrl
